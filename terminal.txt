cd.. -> volta uma pasta/ sobe um nível na estrutura de pastas
cd nomepasta -> entra na pasta
dir -> lista os diretórios da pasta atual
mkdir nomepasta -> cria uma pasta com o nome escolhido no diretório atual
type nul > nomearquivo.js -> cria um arquivo em qualquer extensão desejada.
cls - limpa o terminal
npm install pacote -> instala o pacote desejado
npm remove pacote -> remove pacote
npm install -g nodemon ->atualiza o servidor automaticamente
npm install ejs -> preparar para receber js no html
npm init - criação do package.json
